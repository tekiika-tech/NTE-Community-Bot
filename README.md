# NTE-Community-Bot
## 使用技術

| 技術 | 使用用途 |
|------|----------|
| Python | Bot本体の開発言語 |
| discord.py | Discord Botの作成・スラッシュコマンド・イベント処理 |
| SQLite | イベント情報などのデータ保存 |
| python-dotenv | Botトークンなどの環境変数の管理 |
| Git | ソースコードのバージョン管理 |
| GitHub | ソースコードの共有・バックアップ・共同開発 |
| Visual Studio Code | 開発用エディター |
| WSL (Windows Subsystem for Linux) | Linux環境での開発・実行 |
